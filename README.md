# Dousearch v2.2

**Dousearch** is a powerful Chrome extension designed for digital content creators and researchers. It allows you to search for videos on **TikTok** and **Douyin (Chinese TikTok)** directly, bypassing login requirements and regional restrictions.

## 🚀 Features
* **Cross-Platform Search:** Quickly toggle between Global TikTok and Douyin.
* **No Login Required:** Search and browse content without needing an account.
* **Optimized for Creators:** Designed to help find trending topics and viral content across different regions.
* **Modern UI:** Updated interface for faster navigation and better user experience.

## 📂 Installation (Manual Load)
Since this is a developer version, follow these steps to install:
1. Download this repository as a ZIP file or get the ZIP from the Releases section and extract it.
2. Open your Chrome browser and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right toggle).
4. Click **Load unpacked** and select the folder containing `manifest.json`.

## 🛠️ Tech Stack
* **Manifest V3**
* **JavaScript / CSS / HTML**
* **Declarative Net Request API** (for advanced search handling)
